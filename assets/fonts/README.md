# Drop the licensed TT Fors webfonts here

Place the purchased TT Fors `.woff2` files in this folder with exactly these names
(the `@font-face` rules in `css/styles.css` already point at them):

```
TTFors-Regular.woff2    (weight 400)
TTFors-Medium.woff2     (weight 500)
TTFors-DemiBold.woff2   (weight 600)
TTFors-Bold.woff2       (weight 700)
TTFors-ExtraBold.woff2  (weight 800)
```

Until these files exist, the site renders in Onest (free Google font) — no code
changes needed either way. Only commit the font files if the TypeType license allows it.
